# Hello VuePress

Little Vuepress show-case

## Setup

Vuepress requires Node.js 8.6 or newer.

````
yarn global add vuepress
# OR npm install -g vuepress
````

# create a markdown file

```
echo '# Hello VuePress' > README.md
```


## Startup

###  start writing
vuepress dev

### build to static files
vuepress build
